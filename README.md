## Bolo de Fatia

Essa é uma aplicação mobile para uma loja e-commerce de uma confeitaria chamada **Bolo de Fatia**. Ela foi desenvolvida em React Native com um backend postgresql hospedado no supabase.

## Dependências Principais

- **expo** — framework e plataforma de desenvolvimento
- **react** — biblioteca base do React
- **react-native** — framework mobile
- **typescript** — tipagem estática
- **react-navigation** - navegação entre páginas
- **supabase** - cliente do Supabase (PostgreSQL)

## Como começar

1. Instale as dependências necessárias

   ```bash
   npm install
   ```

2. Inicie o app

   ```bash
   npx expo start
   ```
